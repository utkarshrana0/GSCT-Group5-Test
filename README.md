# GSCT Group 5's Test-over-Cloud framework

Automated testing (using cloud platform) of Group 5's GSCT Recipe Generator Application.



## Technology Stack

 - [WebdriverIO](https://webdriver.io/)
 - [Javascript ES6](https://www.javascript.com/)
